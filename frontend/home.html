How It Works

01
HR Inputs Data
HR enters employee details through the clean, validated frontend form. All fields are checked before submission.
02
API Processes Request
Express.js receives the request, validates the payload, and routes it through the MVC controller layer.
03
MongoDB Persists
Mongoose stores the data in MongoDB Atlas. Auto-generated IDs, timestamps, and schema validation fire automatically.
